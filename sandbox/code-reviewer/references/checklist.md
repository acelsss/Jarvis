# 代码审查检查清单

## 通用检查项

### 代码结构
- [ ] 函数和类职责单一
- [ ] 代码模块化程度适当
- [ ] 依赖关系清晰
- [ ] 避免深度嵌套

### 命名规范
- [ ] 变量和函数名具有描述性
- [ ] 遵循语言特定的命名约定
- [ ] 避免缩写和模糊命名
- [ ] 常量使用大写

### 错误处理
- [ ] 异常处理完整
- [ ] 错误信息清晰有用
- [ ] 资源正确释放
- [ ] 边界条件处理

### 性能考虑
- [ ] 避免不必要的计算
- [ ] 适当使用缓存
- [ ] 数据结构选择合理
- [ ] 算法复杂度可接受

## 语言特定检查

### Python
- [ ] 遵循PEP 8规范
- [ ] 正确使用列表推导式
- [ ] 避免全局变量
- [ ] 适当使用生成器

### JavaScript
- [ ] 避免全局变量污染
- [ ] 正确处理异步操作
- [ ] 避免回调地狱
- [ ] 使用严格模式

### Java
- [ ] 遵循Java命名约定
- [ ] 正确实现equals和hashCode
- [ ] 适当使用设计模式
- [ ] 内存泄漏检查

## 安全检查

- [ ] 输入验证和清理
- [ ] SQL注入防护
- [ ] XSS防护
- [ ] 敏感信息保护
- [ ] 权限控制正确
