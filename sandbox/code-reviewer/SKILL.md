# 代码审查器

本技能提供全面的代码审查和分析能力，帮助识别代码中的问题并提供改进建议。

## 核心功能

### 代码质量检查
- 代码结构和组织评估
- 命名规范检查
- 代码复杂度分析
- 可读性和维护性评估

### 潜在Bug检测
- 空指针和边界条件检查
- 逻辑错误识别
- 资源泄漏检测
- 并发问题分析

### 性能问题分析
- 算法效率评估
- 内存使用优化建议
- I/O操作优化
- 缓存策略建议

### 安全漏洞识别
- 输入验证检查
- SQL注入风险
- XSS漏洞检测
- 权限控制评估

## 使用指南

1. **分析单个文件**：直接提供文件路径或代码内容
2. **批量审查**：指定目录路径进行批量分析
3. **特定问题检查**：明确指出需要关注的特定问题类型
4. **生成报告**：提供结构化的审查报告

## 审查流程

1. 理解代码上下文和目的
2. 应用相应的检查清单
3. 识别潜在问题和改进机会
4. 提供具体的、可操作的建议
5. 生成优先级排序的问题列表
