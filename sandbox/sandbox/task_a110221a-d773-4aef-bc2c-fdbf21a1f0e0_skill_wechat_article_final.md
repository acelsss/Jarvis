# 微信公众号文章生成技能

## 功能说明

本技能用于生成符合微信公众号风格的原创文章，包含以下部分：

1. **标题**：吸引人的标题，通常包含数字或疑问句
2. **摘要**：简短的文章摘要，概括核心内容
3. **正文**：结构化的正文内容，包含多个段落
4. **结尾**：总结性结尾，可包含行动号召

## 使用方式

用户只需提供文章主题或关键词，系统将自动生成完整的文章结构。

## 执行步骤

1. 分析用户输入的主题，确定文章类型和风格
2. 生成文章标题（3-5个候选，选择最佳）
3. 撰写文章摘要（100-200字）
4. 生成正文内容（分为3-5个段落，每段200-300字）
5. 撰写结尾段落（包含总结和行动号召）
6. 将完整文章保存为 Markdown 文件
7. 生成文章元数据文件（包含标题、摘要、字数统计等）

## 输出格式

- 文章文件：`{task_id}_article.md`
- 元数据文件：`{task_id}_article_meta.json`

## 注意事项

- 确保内容原创，避免抄袭
- 标题要吸引人但不过分夸张
- 正文要有逻辑性和可读性
- 结尾要有明确的行动号召
